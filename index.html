    Welcome to xml-drafts

    This small Android™-Framework is an easy way to write an XML-parser based on an XmlPullParser. This framework is written by me and open-sourced under Apache license on github. It aims to work with XML and to write less code as possible. Download the source-code and use it as Android-library.

    Getting started

    you need an xml:
          <root>
         <item name="test-name" />
      </root>
    set up your parser-states:
            enum States implements Tag {
          ITEM( "item "),
          ROOT( "root", ITEM)     
         ...
       }
    write a parser based on states
              final XMLTagHandler tagHandler = 
                              new XMLTagHandler(States.ROOT, callback) {

            @Override
            protected void startTag(final Tag current, 
                               final XMLTagFacade facade)
                    throws Exception {

                if( current == States.ITEM  ) {
                        System.out.println( "-- found an item: " 
                           + facade.getAttribute("name" ));
                }
            }

            @Override
            protected void closeTag(Tag current) 
                    throws Exception {
                ;
            }
          };
    run a parser
            final InpuStream in = ...;
        final XMLParserLoop loop = XMLParserLoop.create( tagHandler  );

        loop.useInput( in );
        loop.run();
